# Prioritizing_Potential_Aquaculture
Marine aquaculture offers a sustainable protein source with minimal ocean footprint. This project maps optimal West Coast U.S. EEZ locations for several oyster species and one additional species using sea surface temperature and depth thresholds to identify the most suitable aquaculture areas.
